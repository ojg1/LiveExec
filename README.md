# live-exec README

Welcome to the LiveExec extension! To install it, go into the "Extensions" tab in Visual Studio Code and in the search box, type "LiveExec." Then, press install, and let the rest of your machine do the work.

## Features

LiveExec helps visualize your code editor in a way that changes how you debug. It is a simple yet powerful tool you can use to see your code run in real time!
Features include:
  -Variables changing in code editor
  -Watching your loops run
  -Colorful dynamics.
